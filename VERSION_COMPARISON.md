# Порівняння версій AuditSample Pro: v1.1 vs v2.0

Це оновлення принесло низку ключових покращень інтерфейсу та виправлень помилок для більш стабільної та зручної роботи з програмою.

## Основні зміни у версії 2.0

### 1. Покращення користувацького інтерфейсу (UI/UX)
- **Оптимізовано розмір шрифтів:** Зменшено розмір шрифтів для числових значень (суми, обсяги вибірки тощо) на картках результатів. Це дозволяє коректно відображати великі суми та довгі числа без розривів чи приховування тексту навіть на невеликих екранах.
- **Вирівнювання таблиць результатів:** Таблицю результатів (разом із заголовками та колонками, зокрема «Аудиторська сума») тепер вирівняно та оптимізовано для екрану, щоб запобігти приховуванню крайньої правої частини.
- **Оптимізація візуалізації:** Синхронізована прокрутка таблиць та покращене перемикання вкладок ("Всі елементи вибірки" / "Ключові елементи") для уникнення артефактів відображення.

### 2. Виправлення процесу імпорту
- **Виправлено роботу індикатора завантаження (спінера):** Раніше на стадії "Імпорт даних" спінер зупинявся кілька разів (до появи кнопки "До налаштувань"). У версії 2.0 анімація завантаження працює стабільно та безперервно протягом усього циклу фонової валідації та підготовки даних, покращуючи інформативність процесу для користувача.

### 3. Загальна стабільність та швидкодія
- **Оптимізація коду:** Усунено низку неактивних оголошень, покращено обробку пам'яті під час переходу між кроками процедури аудиту.
- **Надійність експорту:** Вдосконалено систему формування файлу Excel (.xlsx) після проведення аудиторських тестів, що зменшує ймовірність помилок під час збереження великих обсягів даних.
- **Синхронізація версіонування файлів збережень:** Формат збережених проектів (і для аудиторів, і для клієнтів) оновлено до версії конфігурації `2.0`.
